01.external library  - React , ReactDOM  , babel

babel - javascript compiler - translate other languages into javascript

in react we dont use normal javascript
we use an enhanced version of javascript  -  JSX

JSX - javascript XML - same as javascript , but we can write HTML directly in our javascript code

  comment in JSX - {/* we can add values inside the html elements */}

problem JSX - web browser doesnt understand JSX , need to transale JSX into javascript, thats why we use babel external library
babel transale JSX in to Javascript

eg-  <script src="https://unpkg.com/supersimpledev/babel.js"></script>
     // type="text/babel" - tell babel to translate all the codes inside here to normal javascript
     <script type="text/babel">
      //we can use html tag directly to code .
      // we can save this to variable
      const button = <button>hello</button>
     </script>

also we can use DOM to create button in javascript
